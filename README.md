# 🐺 Akela

**Akela** es una aplicación web que estoy desarrollando utilizando **PHP + MySQL**, orientada a la gestión de **grupos scouts**.

Su objetivo es ofrecer una interfaz **sencilla, intuitiva y moderna** que facilite la administración de un grupo scout, incluyendo funcionalidades **CRUD completas** y una **estructura modular** que permita ampliar el proyecto fácilmente.

Este proyecto es el reinicio de una aplicación anterior que presenté como **Trabajo de Fin de Grado (TFG)** y que puedes encontrar en GitHub como [badentracker](https://github.com/Covija97/badentracker).

He decidido reiniciar el proyecto desde cero para **mejorar mis prácticas de desarrollo en PHP**, reforzar la **seguridad de la aplicación** y añadir nuevas funcionalidades como el **inicio de sesión con múltiples usuarios**.

> [!NOTE]
> El objetivo final es que la aplicación pueda desplegarse fácilmente en cualquier servidor y que pueda ser utilizada por cualquier persona que desee gestionar su propio grupo scout.
