# AKELA

Akela es el nombre de mi proyecto de fin de grado superior. He decidido reiniciarlo desde cero con el objetivo de mejorarlo, añadir mayor seguridad y permitir el inicio de sesión con diferentes usuarios.

El proyecto original del que surge esta nueva versión es el que presenté en mi TFG. Puedes encontrarlo en GitHub como [badentracker](https://github.com/Covija97/badentracker).

El objetivo final es que la aplicación pueda desplegarse fácilmente en cualquier servidor y así pueda ser utilizada por cualquier persona que desee hacerlo.
